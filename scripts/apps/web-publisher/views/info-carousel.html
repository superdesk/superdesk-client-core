Create site wizard

User wizard to create site and apply theme

In the Output Settings screen, click on the plus button in top right corner. With thisaction, a modal dialog will open - create site wizard. Within the wizard, just follow steps to create your site and choose theme and apply to your site.

Ok, i got it!

Add new site

Step1: define your site

First tab in the modal dialog is Add Site. Fill in your new Site Name and provide Domain Name, in form www.domain.com. Click continue.

Ok, i got it!

Choose the theme

Step2: Browse through available themes

On the Install Theme tab, you can quickly browse throught available themes list. By clicking on details/install button you can check theme details and install the theme. Preview will open new browser tab with theme demo instance, with predefined content.

Ok, i got it!

Install the theme

Step3: Check theme details and install the theme

Once you entered theme details, you can check additional information about the theme. If you like the theme, you can install it, by clicking on Install button. You can easily get back to themes list by clicking on left arrow.

Ok, i got it!

Activate the theme

Step4: Activate the theme

Once you finish install process, you have to activate the theme to be able to use it on your site.

Ok, i got it!

Configure site or done

Step5: Finish or continue configuring your site

This is last step in the wizard. After you activated chosen theme it should be marked as green. At this point you can finish (click done button), or continue configuring your website. If you click Configure site, main settings dialog will open so you can set routes, navigation etc.
